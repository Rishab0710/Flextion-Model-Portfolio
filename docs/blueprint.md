# **App Name**: FlexAI Support

## Core Features:

- Intelligent Question Answering: Users can type questions into the search bar, and AI provides relevant answers or directs them to appropriate resources, functioning as an AI-powered tool for assistance.
- FAQ Section: Provides quick access to a list of frequently asked questions.
- History Tracking: Logs user questions to assist support personnel, and to provide a history to the user themselves. Save user queries for future reference
- Profile Display: Show a reduced avatar for user indication

## Style Guidelines:

- Primary color: A vibrant, warm orange (#E3733D) evokes creativity and enthusiasm, while fitting the brand. This provides an eye-catching focus point to draw attention without feeling aggressive.
- Background color: Dark, desaturated brown (#383430) provides a sleek, professional look with minimal distraction, ensuring readability and visual comfort.
- Accent color: Analogous to the warm orange but a contrasting vivid mint-green (#75E1B9). Use this to indicate success/confirmation.
- Headline font: 'Space Grotesk' sans-serif for headlines. It lends a tech-focused contemporary vibe.
- Body font: 'Inter' sans-serif for body text. When combined with Space Grotesk for headings, Inter creates a balanced sense of modernity.
- Crisp and professional line icons.
- Clean and minimalistic layout with a centered focus.
